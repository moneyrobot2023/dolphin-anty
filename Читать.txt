
Скрипт для бесконечного количества профилей для бесплатного Dolphin Anty

Сначала вам нужно войти в свою учетную запись в программе Dolphin Anty или создать новую, поскольку 
старые профили браузера все равно не будут видны. Вам нужно убедиться, 
что у вас есть хотя бы 1 незанятый профиль в наличии до истечения лимита профиля (9/10), 
иначе профилирование не будет работать. Затем мы закрываем антидетект.
Скопируйте файл app.asar из папки files скрипта и переместите его с заменой в 
папку с установленным antidetect \Dolphin Anty\resources. Скорее всего, 
вы можете найти нужную папку antidetect по одному из следующих путей:

C:\Users*your username*\AppData\Local\Programs\Dolphin Anty\resources
C:\Program Files\Dolphin Anty\resources
pip install -r requirements.txt
python main.py
Настройки:
Вы можете открыть settings.py файл и установить переменной backup значение True или False. 
Каждый раз, когда профиль успешно закрывается, он будет скопирован 
в папку browsers_backup
